Reports deprecated include and exclude options under flags.warn_error_options in dbt_project.yml.

Since dbt v1.10, these keys have been renamed:

  • include -- renamed to error
  • exclude -- renamed to warn

For example, instead of:


flags:
  warn_error_options:
    include:
      - NoNodesForSelectionCriteria

Use:


flags:
  warn_error_options:
    error:
      - NoNodesForSelectionCriteria

See WEOIncludeExcludeDeprecation for details.
