Reports deprecated configuration keys in dbt_project.yml.

The following keys have been renamed or removed and will cause errors in future dbt versions:

  • data-paths -- renamed to seed-paths (since dbt v1.0)
  • source-paths -- renamed to model-paths (since dbt v1.0)
  • target-path -- removed (since dbt v1.5, use --target-path CLI flag or DBT_ENGINE_TARGET_PATH env var)
  • log-path -- removed (since dbt v1.5, use --log-path CLI flag or DBT_ENGINE_LOG_PATH env var)
  • dbt_modules in clean-targets -- should be dbt_packages

Quick-fixes are available to rename or remove the deprecated keys automatically.

See dbt Deprecations for details.
